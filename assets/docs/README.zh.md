# Pianpker · 落纸之间

> 字落有间，读来无距。

Pianpker 是 [Drag0nM](https://github.com/DRAG0NM) 的个人博客项目。它以文字阅读为中心，用克制的留白、和纸暖色、墨色层级与少量朱红，建立安静而清晰的数字排版体验。

## 设计原则

- 不用表面的“日式”装饰代替排版思考；
- 以侘寂、幽玄、物哀与「间」组织空间和节奏；
- 内容可以复杂，形式尽可能简单；
- 每一个视觉元素都应有存在的理由。

## 技术与功能

项目基于 Astro 6 与 UnoCSS，支持多语言内容、明暗主题、ClientRouter、Markdown/MDX、KaTeX、Mermaid、目录、图片处理、RSS、Open Graph、响应式布局与打印。

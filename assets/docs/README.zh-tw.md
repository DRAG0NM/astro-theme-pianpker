# Pianpker · 落紙之間

> 字落有間，讀來無距。

Pianpker 是 [Drag0nM](https://github.com/DRAG0NM) 的個人部落格專案。它以文字閱讀為中心，用克制的留白、和紙暖色、墨色層次與少量朱紅，建立安靜而清晰的數位排版體驗。

## 設計原則

- 不用表面的「日式」裝飾取代排版思考；
- 以侘寂、幽玄、物哀與「間」組織空間和節奏；
- 內容可以複雜，形式盡可能簡單；
- 每一個視覺元素都應有存在的理由。

## 技術與功能

專案基於 Astro 6 與 UnoCSS，支援多語言內容、明暗主題、ClientRouter、Markdown/MDX、KaTeX、Mermaid、目錄、圖片處理、RSS、Open Graph、響應式佈局與列印。
